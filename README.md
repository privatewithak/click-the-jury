## simple clicker 
### features:
- click the uu
- progress through the cards (level progression)
- buy shop upgrades:
  - "UNION worker" — +1 auto-click per second
  - "citizen slave" — +1 click power per purchase
- crit frenzy: rare temporary double-power window (implemented in-game)
- sound effects and particle bursts on click
- combo counter (chain clicks within a short window)
- local autosave (progress saved to localStorage)
- access an in-game updatelog from the card UI

### stack:
- react + swc
- tailwindcss
- framer motion
